Section 1: HTML / CSS 기초

220428- HTML / CSS 기초

Achievement Goals

  • HTML에 CSS를 적용하고, JavaScript로 개발할 수 있게(Progtammable) 작성
  • HTML이 Markup Language 라는 것을 이해
    • “구조를 표현하는 언어” 라는 의미
  • HTML의 구조와 문법에 대해 이해
    • Openig tag, closing tag, self-closing tag
  • 자주 사용되는 HTML 요소(element)
    • div, span tag의 용도 및 차이
    • ul, ol, li
    • input type을 설정하고 다양한 종류의 input 사용
  • 동적인 웹 어플리케이션 개발을 위한 HTML 구조 짜기
    • 간단한 웹 페이지 기획안을 HTML 문서로 표현
    • id와 class를 목적에 맞게 사용하여 사람과 컴퓨터가 읽기 쉬운, 의미 있는(Sementic) 문서를 작성
    • HTML5 semantic tag를 적재적소에 사용하여 시멘틱한 HTML 문서 작성

HTML?

  • HyperText Markup Language
  • 웹 페이지의 틀을 만드는 마크업 언어

사용법

  • Html은 tag들의 집합
  • Tag : 부등호(<>)로 묶인 기본 구성 요소
  • html 확장자 사용

TREE STURCTURE

  • HTML 문서 시작 →
    • html → 시작 태그, 문서 전체의 틀 구성
      • head → 문서의 메타데이터 선언
        • title : Page title, 문서의 제목 → 브라우저 탭에 노출
      • body → 문서의 내용
        • h1 → heading을 의미, 크기에 따라 h1~h6
        • div : Content division → 줄바꿈 됨
          • span : Content Container → 줄바꿈 없음
